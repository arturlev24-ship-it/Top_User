# 🏆 Telegram Rating Bot

Бот для создания рейтинга участников в Telegram чатах. Начисляет очки за активность и показывает топ-10 пользователей.

## ✨ Возможности

- 📝 Автоматическое начисление очков за каждое сообщение в чате
- 🏆 Отображение топ-10 участников по общему рейтингу
- 💬 Подсчет очков со всех чатов, где есть бот
- 📊 Отображение личной статистики и позиции в рейтинге
- 🚀 Асинхронная работа с базой данных (aiosqlite)
